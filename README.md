# HOW TO RUN
Firstly you have to install node module in client and server directory separately

Server Node Install :
1. cd eric-tube
2. npm install

Next,install in client also:
1. cd client/
2. npm install

Then you go back to previous directory or in server 
1. cd ..

finally you can run the web app
1. npm run dev
